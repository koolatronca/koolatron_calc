# Koolatron Smart Calculator

Touch-friendly scientific calculator for robot/kiosk browser use.

Flow improvements:
- Type `30`, press `sin`, and it becomes `sin(30)`.
- Type `5`, press `x²`, and it becomes `(5)²`.
- Type `2`, press `xʸ`, then `3`, and it becomes `(2)^3`.
- Functions wrap the previous value automatically.
- Missing closing brackets are auto-closed during calculation.
- DEG/RAD toggle included.
